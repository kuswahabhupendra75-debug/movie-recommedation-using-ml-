# CineHybrid - Movie Recommendation System

CineHybrid is a comprehensive movie recommendation system leveraging machine learning to provide highly personalized movie suggestions. 

## Features

- **Personalized Recommendations:** Get tailored movie suggestions based on your preferences.
- **Modern User Interface:** A sleek and responsive frontend built with web technologies for an optimal user experience.
- **Robust Backend:** Powered by Python for efficient machine learning processing and API endpoints.
- **Trending & Popular:** Discover the top-rated and trending movies from our vast catalog.

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript, React/Vite (or Next.js)
- **Backend:** Python, Flask/FastAPI
- **Data ML:** Scikit-learn, Pandas

## Project Structure

- `/frontend` - Contains the user interface and client-side code.
- `/backend` - Contains the server-side code, ML models, and API logic.
- `/data` - Datasets and CSVs used for training and recommendations.
- `.github` - GitHub Actions for automated workflows and continuous deployment.

## Running Locally

1. **Backend:**
   Ensure Python is installed. Navigate to the `backend` directory, install dependencies from `requirements.txt`, and run the backend server.
   
2. **Frontend:**
   Navigate to the `frontend` directory, install dependencies (`npm install`), and start the development server (`npm run dev`).

## Deployment

The backend is configured for deployment on Render (`render.yaml`), and the frontend can be deployed on Vercel (`vercel.json`).
